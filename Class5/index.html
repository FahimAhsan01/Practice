CSS:

CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Types of CSS:

  • Inline CSS:

    Inline CSS is used to apply a unique style to a single HTML element. It is applied using the style attribute of the HTML tag. For example:

                    <h3 style="color: red; text-align:center">This is a heading</h3>
  • Internal CSS:

    Internal CSS is used to apply a unique style to a single HTML page. It is applied using the style tag in the head section of the HTML page. For example:

                    <style>
                    h3 {
                                    color: red;
                    }
                </style>
  • External CSS:

    External CSS is used to apply a unique style to multiple HTML pages. This is the most preferred and commonly used method for applying CSS to html. It is applied using a separate CSS file. The external CSS file is linked to the html file using the following statement.

                    <link rel="stylesheet" href="styles.css">

Selectors, Properties and Values in CSS:

Selector: A selector is an HTML tag at which a style will be applied. This could be any tag like h1, h2, div etc.

Properties: Properties are the styles that you can apply to the selector. For example, color, font-size, background-color etc.

Values: Values are the assigned values to the properties. For example, if the property is color, the value could be red, green, blue etc.

Example:

Let's say we want to change the color of all the h1 tags to red. The CSS code for this would be:

        h1 {
                        color: red;
        }

Here, h1 is the selector, color is the property and red is the value.

There are 3 kinds of selectors and each one of them have their own calls. These are:

  • Element Selector
  • Class Selector
  • ID Selector

Element Selector: This selects all the elements of a particular type. For example, if we want to select all the h1 tags, the selector would be

h1

Class Selector: This selects all the elements with a particular class. For example, if we want to select all the elements with the class name "heading", the selector would be

.heading

ID Selector: This selects a single element with a particular ID. For example, if we want to select the element with the ID "main-heading", the selector would be

#main-heading

The ID selector has the highest priority, followed by the Class selector and finally the Element Selector. If a single selector is affected by multiple properties and values, then the value at the last of the list will be the one that will remain active.
Only a single ID selector can be used per element but there is no limit to the number of Class selectors. So, multiple Class selectors can be used for a single element to carry over previous CSS styles and customize a new one.

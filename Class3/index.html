Anchor tag:

The anchor tag or <a> tag is used to hyperlink external links to a text. The attribute target="_blank" enables new tab opening.

For example, here's a link directing to the page detailing this tag: Anchor tags in detail

Inline Frame tag:

The inline frame tag or <iframe> tag creates an interactable frame within the current html page to display or utilize contents found in the internet allowed to be shared by the owners. This can be anything from youtube videos and google maps locations to full blown websites and documents.

Video and Audio tags:

The video and audio tags are used to embed videos and audios in the html page. The controls attribute enables the user to play, pause, and adjust the volume of the media.
Note that a video file with mp4 format can be easily converted to an audio file just by changing the file extension to mp3. This process is irreversible though.

Marquee tag

The marquee tag is used to create a scrolling text or image. The attributes direction="up", "down", "left", "right" and behavior="scroll", "slide", "alternate" can be used to customize the scrolling effect.

Forms:

Finally we have the forms which are used to collect user input. The <form> tag is used to create a form. The input tag or <input> tag is used to create input fields within the form. The type attribute of the input tag is used to specify the type of input field. The value attribute of the input tag is used to specify the default value of the input field. The name attribute of the input tag is used to specify the name of the input field. The placeholder attribute of the input tag is used to specify a hint to the user about what to enter in the input field. The required attribute of the input tag is used to specify that the input field must be filled out before submitting the form. The form tag has an action attribute which specifies the URL of the page the form is submitted to. The form tag has a method attribute which specifies the HTTP method used when submitting the form.
